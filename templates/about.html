Hướng dẫn sử dụng:
  1. Synthetic Benchmark: Chọn số lượng phần tử, thuật toán hash, nhấn "Run Synthetic" để so sánh hiệu năng các thuật toán.
  2. Realistic Benchmark: Upload file CSV (key,value), chọn thuật toán, nhấn "Upload & Run Benchmark" để đo hiệu năng trên dữ liệu thực tế.
  3. Fast Feature Map: Nhập chuỗi text, chọn thuật toán hash, nhấn "Vectorize" để xem ánh xạ feature → id và vector 1-hot.
  4. CRUD Demo: Thực hành Insert/Search/Delete trực tiếp trên các hash table.
  5. Xuất kết quả: Sau khi benchmark, nhấn "Xuất CSV" để tải về kết quả so sánh.
